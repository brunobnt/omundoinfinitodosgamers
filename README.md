# O Mundo Infinito dos Gamers

Um estudo sobre o consumidor de games no Brasil. Mergulhe na complexidade desse universo, conheça os perfis de gamers e entenda como marcas podem ser mais relevantes.

- **Disponível em:** [https://omundoinfinitodosgamers.com.br/](https://omundoinfinitodosgamers.com.br/)

## Como executar?

Antes de executar essa aplicação, você precisará instalar o [Node.js](https://nodejs.org/en/) e o [NPM](https://www.npmjs.com/) em seu computador. Após isso, siga o tutorial a seguir:

##### 1. Usando o comando abaixo, clone este repositório para o seu computador.

```sh
git clone https://github.com/brunobnt/omundoinfinitodosgamers.git
```

##### 2. No terminal, navegue até o diretório raíz do repostório local.

```sh
cd omundoinfinitodosgamers
```

##### 3. Agora, instale as dependências da aplicação usando o NPM.

```sh
npm install
```

##### 4. Inicie a aplicação (em modo de desenvolvimento) usando o comando abaixo.

```sh
npm run dev
```

##### [EXTRA] Se precisar executar a aplicação em modo de produção, utilize os comandos abaixo.
```sh
npm run build
npm run preview
```
<br/>

Após iniciada, a aplicação poderá ser acessada em **`http://localhost:5173/`** usando um navegador web de sua preferência.

Para encerrá-la, retorne ao terminal e pressione **`Ctrl + C`**.
